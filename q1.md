## Github的作用：

GitHub 是全球最大的基于 Git 的代码托管与协作平台，主要用于开发者存储、管理、共享代码，同时也是开源项目交流和技术社区，相当于程序员们的小红书。
- Pull request 
简称PR，相当于你向代码仓库的维护者提交的 “修改申请” 
- Fork
就是把别人 GitHub 仓库（比如开源项目）完整 “复制” 一份到你自己的 GitHub 账号下，复制后你拥有这个副本的完全控制权（可修改、提交、删除），且不影响原仓库的代码。
