# Github学习笔记

## 打开方式

1.打开windows的自带工具watt tooikit
2.点击网络加速，找到github并选择
3.打开任意浏览器Edge或者Chrome 搜索github官网就能进去了

## 学习平台

>https://docs.github.com/en/get-started/start-your-journey/hello-world

通过图文指引手把手教你创建仓库、提交更改、体验分支操作，直观感受 GitHub 核心流程

>https://www.bilibili.com/video/BV1hS4y1S7wL/?share_source=copy_web

视频教程，简单易懂了解github上的基本操作

>https://git-scm.com/book/zh/v2

从 Git 安装、核心命令（git init/git add/git commit 等）到工作原理全面讲解，配有图解和示例，既是入门教材也是后续查阅的参考手册
